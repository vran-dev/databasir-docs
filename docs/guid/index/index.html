# Databasir

是面向团队的关系型数据库模型文档管理平台，旨在通过自动化的方式解决模型文档管理过程中维护成本高、内容更新不及时以及团队协作复杂等问题。

问题反馈：https://github.com/vran-dev/databasir/issues

# 支持平台

理论上支持拥有 JDBC 驱动的任何数据库，以下是社区内已验证过的数据库

  • MySQL
  • PostgreSQL
  • Oracle
  • SqlServer
  • 达梦
  • TIDB
  • Hive
  • Clickhouse
  • MariaDB
  • .....

注意：以上列表是目前社区已经验证过的，并不代表 Databasiropen in new window 只支持这些。
